# AI Chatbot Project

A Flask-based AI data generator that converts natural language requirements into SQL queries.

## Output Preview

![AI Data Generator output](ai-chatbot/assets/app-screenshot.png)

## Run Locally

```powershell
cd ai-chatbot
python -m pip install -r requirements.txt
python app.py
```

Open the app at:

```text
http://127.0.0.1:5000
```
